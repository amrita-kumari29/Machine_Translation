Language Detection and Translation System

 # Project Overview
This project detects the language of an input text using Machine Learning and NLP techniques and automatically translates the detected language into English.

 # Features
- Language detection using Naive Bayes Classifier
- NLP-based text vectorization (CountVectorizer)
- Automatic language translation
- Supports multiple languages including Hindi, French, Spanish, German, etc.

 # Tech Stack
- Python
- NumPy, Pandas
- Scikit-learn
- NLP (CountVectorizer)
- Googletrans (Translation API)
